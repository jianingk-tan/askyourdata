### Products.Discontinued

Declared type: TEXT. Distinct values: ['0', '1']

**Quirk:** Stored as text `'0'` / `'1'`, not boolean. The agent must filter with `Discontinued = '1'` (string), not `Discontinued = 1` (integer).

### UnitPrice across tables

`Products.UnitPrice` samples: [18, 19, 10, 22, 21.35]
`Order Details.UnitPrice` samples: [14, 9.8, 34.8, 18.6, 42.4]

**Quirk:** Products holds the *list price* (integer-ish), Order Details holds the *transaction price* (can be fractional due to historical pricing). These are intentionally different — not normalized.

### Orders.OrderDate

Declared as DATETIME, stored as ISO 8601 strings. SQLite handles `strftime()` and `date()` correctly on these. No action needed.

### Order Details.Discount

**Quirk:** Discount is a decimal (0.10 = 10%), not a percentage (10 ≠ 10%). The agent must multiply, not subtract.
